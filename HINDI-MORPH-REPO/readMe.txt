how to run the code
python run_morph_on_file_with_raw_text.py --input inputFile --output outputFile

Pre-Requisites to run this code

1. Install the morph analyzer, go to the morph-hin-le-5.0.5 folder and follow the instructions inside the INSTALL file [runs on Ubuntu 18.04]
2. Install libglib, libgdbm
sudo apt install libgdbm-dev libglib2.0-0
3. This program is designed in such a way that it produces the morph analysis of each sentence present in a word problem
